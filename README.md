# README
#### 这是一个简单的你画我猜项目,基于Konva,Rails 5实现
#### websocket使用rails自带的actioncable
#### 使用方式
* 安装redis
* bundle
* yarn
* bundle exec rails db:migrate
* rails s

# 项目依赖

* Ruby version 2.4.1

* Ruby on Rails

* redis

* node v 8.15.0

* yarn 1.13.0

# feature

* 聊天表情

* 分配房间

* 画图系统优化,加入更多颜色画笔

* 优化画图同步逻辑,改善卡顿

* 用device实现一个成熟账号系统

* 账号积分

* 历史绘画

* 分类词库可选词库更多词库自定义词库

# 已实现

* 简单聊天系统

* 简单词库
